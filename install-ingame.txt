QUICK INSTALLATION FOR WINDOWS INGAME SERVER
--------------------------------------------

For first install use the fast3.2.x-winphp5.zip package !

You are supposed to already have a fonctionnal TM server

- Extract the files in any directory (eventually your game one)

- Modify 'ingame.cfg' with wordpad or any other editor, you have
  to set the values for :

	Server TMF login in:        <login></login>
  Server TMF password(*) in:  <password></password>

  If you changed the default ports of your game server then
  you have also to modify them in the file.

- Modify if you wish fast.php with wordpad or some other text editor
  to adapt the votes etc. to your needs.

- start 'FAST3 ingame.bat'


(*)or community code !

NOTE: your server HAVE TO really use the ports indicated in the
game config, so those configured ports must not be used
by another program !!!
PS: You can get and use the tool TCPView (search on google) to see what
program use what port easily.

(*)ou community code !
